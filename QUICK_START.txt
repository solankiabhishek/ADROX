ADROX E-Commerce Website - Quick Start Guide
============================================

🚀 GETTING STARTED

1. LOCATE FILES
   - Extract all files to: C:\xampp\htdocs\adrox\

2. START SERVICES
   - Open XAMPP Control Panel
   - Click "Start" for Apache
   - Click "Start" for MySQL

3. INITIALIZE WEBSITE
   - Open browser
   - Go to: http://localhost/adrox/setup.php
   - Wait for setup to complete

4. ACCESS WEBSITE
   - Main Website: http://localhost/adrox/
   - Admin Panel: http://localhost/adrox/admin/
   - Admin Password: admin / admin123

🎯 ADDING YOUR FIRST PRODUCT

1. Go to http://localhost/adrox/admin/
2. Login with: admin / admin123
3. Click "Add New Product"
4. Fill in details:
   - Product Name: e.g., "Premium Mobile Cover"
   - Description: e.g., "Red luxury mobile cover with protection"
   - Price: e.g., 299
   - Category: e.g., "Mobile Covers"
   - Affiliate Link: Your affiliate link
   - Mark as Featured: Check if you want it on homepage
5. Click "Save Product"

The product will now appear on your website!
